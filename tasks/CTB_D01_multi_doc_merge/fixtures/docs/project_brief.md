# North Star Store Analytics Project

## Background

The client wants to establish a unified store operations analytics capability for their offline retail business, starting with a pilot before deciding whether to roll out company-wide.

## Goals

- Pilot covering **120 stores**
- Deliver a unified store operations dashboard
- Provide anomaly alerting, prioritizing inventory anomalies and promotion conversion anomalies
- Generate a weekly operations summary for management

## Preliminary Timeline

- Original planned kickoff: **2026-04-01**
- Pilot duration: 6 weeks

## Notes

The client prefers read-only integration first and does not want the pilot phase to write back to the POS system.
